Siglas

OSI

Modelo OSI (Open Systems Interconnection) es un marco conceptual que se utiliza para entender y describir cómo los diferentes componentes de un sistema de comunicación de red interactúan.

IP

El Protocolo de Internet (IP) es un protocolo de comunicaciones utilizado para identificar dispositivos en una red y enrutar paquetes de datos a su destino.

TCP

El Protocolo de Control de Transmisión (TCP) es uno de los principales protocolos de comunicación en la capa de transporte de la red, asegurando la entrega confiable de datos entre dispositivos.

UDP

El Protocolo de Datagramas de Usuario (UDP) es un protocolo de la capa de transporte que permite la transmisión de datos sin garantizar su entrega, utilizado en aplicaciones donde la velocidad es más importante que la confiabilidad.

SSH

SSH (Secure Shell) es un protocolo de red utilizado para acceder de forma segura a dispositivos remotos, asegurando la comunicación cifrada entre dos sistemas.

FTP

FTP (File Transfer Protocol) es un protocolo utilizado para transferir archivos entre dispositivos en una red.

SMTP (Simple Mail Transfer Protocol)

SMTP es un protocolo de red utilizado para el envío de correos electrónicos a través de Internet.

POP

POP (Post Office Protocol) es un protocolo utilizado para la recepción de correos electrónicos desde un servidor a un cliente de correo, permitiendo descargar los mensajes.

DHCP

DHCP (Dynamic Host Configuration Protocol) es un protocolo que permite a los dispositivos en una red obtener automáticamente una dirección IP y otros parámetros de configuración de red.

DNS

DNS (Domain Name System) es el sistema que traduce nombres de dominio legibles por humanos a direcciones IP necesarias para la comunicación en redes.

HTTP (Hypertext Transfer Protocol)

HTTP es un protocolo de comunicación utilizado para la transferencia de documentos web a través de la red. Es la base de la web.

ARP

ARP (Address Resolution Protocol) es un protocolo utilizado para mapear direcciones IP a direcciones MAC en una red local.

MAC

MAC (Media Access Control) es una dirección única asignada a una interfaz de red para identificar dispositivos dentro de una red local.

LAN

LAN (Local Area Network) es una red que conecta dispositivos dentro de un área geográfica limitada, como una casa, oficina o campus.
